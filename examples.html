The dmslogo package is designed to aid in the visualization of deep mutational scanning via line and logo plots of site- and mutation-wise measurements of mutationsl effects. It provides several Python functions that you can use in scripts or Jupyter notebooks. Major functionality includes:

  • The dmslogo.logo.draw_logo function for creating highly customizable logo plots.

  • The dmslogo.line.draw_line function for creating line plots.

  • The dmslogo.facet.facet_plot function for faceting together combinations of line and logo plots.
